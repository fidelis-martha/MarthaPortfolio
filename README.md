# 🌟 Martha Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, experience, and journey as a Front-End Developer, Web3 Enthusiast, and Content Creator.

## 🚀 Live Demo

🔗 [View Portfolio](https://fidelis-martha.github.io/MarthaPortfolio/)

## 📌 Features

- Responsive design for mobile, tablet, and desktop devices
- Modern and clean user interface
- About Me section
- Skills showcase
- Project gallery with project details
- Contact section
- Downloadable resume/CV
